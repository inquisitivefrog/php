# Laravel Features Examined

This document outlines the Laravel features included in the Laravel Feature Lab project.

## Authentication
- **Breeze** — Minimal auth scaffolding (API, Blade, Inertia, React/Vue)

## Feature Flags
- **Pennant** — Gradual rollouts, A/B testing, blue/green deployment, per-user/team feature gates

## Subscription Billing
- **Cashier (Stripe)** — Subscription management, metered billing, SCA compliance

## Debugging & Monitoring
- **Telescope** — Real-time request, command, queue, exception, DB, and cache monitoring
- **Log Viewer** — Browse Laravel logs in the browser

## Queues
- **Horizon** — Queue dashboard and worker management

## API & Data
- **Sanctum** — Lightweight API token authentication
- **Scout** — Search abstraction layer for various backends

## Code Quality
- **Pint** — Modern code formatter
- **PHPStan / Larastan** — Static analysis

## Testing
- HTTP tests
- Database transaction handling
- Mocking support
- Pest support
